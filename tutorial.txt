__________Как начать Django проект_____________________________

1) python -m venv название вир. окружения - создание вир. окружения
2) название вир. окружения\Scripts\activate.bat - запуск вир. окружения
3) pip install Django - установка Django
4) django-admin startproject название проекта - создание нового проекта
5) cd название проекта - переход к проекту
6) python manage.py runserver - запуск проекта
7) ссылка http://127.0.0.1:8000/
